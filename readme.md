# is-prime [![Build Status](https://travis-ci.org/lborgav/is-prime.svg?branch=master)](https://travis-ci.org/lborgav/is-prime)

> Test if a number is prime


## Install

```
$ npm install --save is-prime
```


## Usage

```js
var isPrime = require('is-prime');

isPrime(2);
//=> true

isPrime(7);
//=> true

isPrime(100);
//=> false
```
